# 🎯 Task Manager Frontend (React)

Frontend application for a task management system built with React.  
Consumes a .NET REST API and provides a clean and responsive user interface for task management.

---

## 🚀 Features

- User authentication (login & register)
- Task creation, editing, and deletion
- API integration with backend (.NET)
- State management
- Form validation
- Responsive UI

---

## 🛠️ Tech Stack

- **Frontend:** React
- **Language:** JavaScript
- **Styling:** CSS
- **API Communication:** Fetch / Axios
